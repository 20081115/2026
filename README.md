# 數學練習遊戲

這是一個可以在瀏覽器中玩的數學練習遊戲，支援：
- 加減法
- 乘除法
- 混合題型
- 多種難度等級

## 使用方式

1. 開啟專案資料夾：`/workspaces/2026`
2. 直接開啟 `index.html`，或用靜態伺服器在瀏覽器中執行
3. 按「開始練習」選擇等級與題型
4. 輸入答案並提交

## 檔案說明

- `index.html`：遊戲網頁
- `style.css`：網頁樣式
- `script.js`：遊戲邏輯

## 本機啟動（可選）

如果希望用簡單的本機伺服器開啟，執行：

```bash
cd /workspaces/2026
python3 -m http.server 8000
```

然後在瀏覽器中開啟 `http://localhost:8000`
