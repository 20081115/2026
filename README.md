# 數學練習遊戲

這是一個可以在瀏覽器中玩的數學練習遊戲，支援：
- 10 種難度等級
- 多種主題：加法、減法、乘法、除法、四則混合、乘法表
- 固定 10 題測驗模式
- 即時答題反饋與得分記錄

## 使用方式

1. 開啟專案資料夾：`/workspaces/2026`
2. 直接開啟 `index.html`，或用靜態伺服器在瀏覽器中執行
3. 選擇難度與主題
4. 點擊「開始 10 題測驗」
5. 每題輸入答案並提交，完成 10 題後可看到成績

## 檔案說明

- `index.html`：遊戲網頁
- `style.css`：網頁樣式
- `script.js`：遊戲邏輯

## 本機啟動（可選）

如果希望用簡單的本機伺服器開啟，執行：

```bash
cd /workspaces/2026
python3 -m http.server 8000
```

然後在瀏覽器中開啟 `http://localhost:8000`
